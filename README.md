# Student Performance Predictor

## Project Overview

Student Performance Predictor is a Machine Learning and Flask-based web application that predicts student performance based on input features. The project uses trained machine learning models and provides predictions through a simple web interface.

## Features

* Student performance prediction
* Flask web application
* Trained machine learning model
* Data visualization support
* CSV dataset integration

## Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* HTML

## Project Structure

* app.py – Main Flask application
* train.py – Model training script
* train_polynomial.py – Polynomial model training
* predict.py – Prediction logic
* visualization.py – Data visualization
* dataset/student_data.csv – Dataset
* model/ – Saved trained models
* templates/index.html – User interface

## Installation

1. Clone the repository.
2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

python app.py
